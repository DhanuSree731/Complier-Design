### **Project Overview – Compiler Visualization Tool**

The **Compiler Visualization Tool** is an interactive web application developed using **Python** and **Streamlit** to demonstrate the complete compilation process of a simple C-like program. It helps users understand how a compiler converts source code into machine-understandable instructions by visualizing each compiler phase.

The application accepts C-like code as input and processes it through different stages, including **Lexical Analysis**, **Syntax Analysis**, **Semantic Analysis**, **Intermediate Code Generation**, **Code Optimization**, **Control Flow Graph (CFG) Generation**, and **Pseudo Assembly Code Generation**. Each stage displays its output in an easy-to-understand visual format, making compiler concepts simple and practical.

The project also provides interactive features such as **token tables**, **parse tree visualization**, **symbol tables**, **three-address code (TAC)**, **optimized code**, **compiler statistics**, and **downloadable CSV/JSON reports**. The modern Streamlit interface allows users to experiment with different programs and instantly observe how each compiler phase works.

This project is particularly useful for **Computer Science students**, **compiler design laboratories**, **faculty demonstrations**, and **educational purposes**, as it bridges the gap between theoretical compiler concepts and practical implementation.

### **Key Highlights**

* Interactive Streamlit web application
* Step-by-step visualization of compiler phases
* Parse Tree (AST) and Control Flow Graph (CFG)
* Symbol Table generation
* Three Address Code (TAC) generation
* Code optimization techniques
* Pseudo Assembly code generation
* CSV and JSON report export
* User-friendly interface for learning compiler design
